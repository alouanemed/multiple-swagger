# Multi Swagger example


## Building and Running
1- Setup the app

```sh
cd plutusSwagger
npm init
npm install express --save
npm install --save swagger-jsdoc swagger-ui-express
```

2- Add seperate yaml files to ```/docs```.

3- Run the app

```sh
node app.js
```

4- Navigate to  ```localhost:3000/docs ```  see the specs
